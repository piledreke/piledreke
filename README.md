## projects

### [tbank-logo-detector](https://github.com/piledreke/tbank-logo-detector)

REST API для детекции логотипа Т‑Банка на изображениях. Inference на ONNX, готов к продакшену.

REST API for T-Bank logo detection on images. ONNX inference, production-ready.

`Precision 0.946` · `Recall 0.932` · `F1 0.939` @ IoU 0.5

### [MoexTradingAgent](https://github.com/piledreke/MoexTradingAgent)

Аналитический агент для MOEX: собирает рыночные данные, считает сигналы, объясняет их через LLM.

Analytical agent for MOEX — market data, composite scoring, LLM explanations.

```
collectors → DuckDB → scoring → LLM → FastAPI
```
